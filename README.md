# AI-Based Stress and Emergency Detection Using Wearable Sensor Data

This project focuses on analyzing wearable IoT sensor data using data science and machine learning techniques to detect stress and emergency conditions. The system aims to support continuous and automatic health monitoring without manual intervention.

---

## ðŸ”¹ Dataset Description
The dataset consists of data collected from wearable sensors, including:
- Accelerometer (a_x, a_y, a_z)
- Gyroscope (g_x, g_y, g_z)
- Temperature sensors
- Heart rate and physiological signals

The data captures human activity and health-related parameters such as movement, posture, and body temperature.

---

## ðŸ”¹ Methodology
The project follows a standard data science workflow:
1. Data collection from wearable sensor datasets
2. Data cleaning (handling missing values, outliers, and formatting)
3. Exploratory Data Analysis (EDA) using visualizations
4. Feature selection based on correlation analysis
5. Machine learning model training
6. Model validation and result analysis

---

## ðŸ”¹ Machine Learning Models Used
The following models were trained and evaluated:
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest

Among these, Random Forest achieved the best performance.
